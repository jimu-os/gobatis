`GoBatis` 是参考 `MyBatis` 编写的sql标签解析，`GoBatis`仅提供对 mapper 的上下文数据解析填充，并不保证对 sql 语句的语法检查,支持自定义数据映射
，详情请[参考文档](https://aurora-engine-cn.github.io/orm/gobatis.html)
